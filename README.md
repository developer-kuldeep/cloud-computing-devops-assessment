# DevOps and Cloud Computing Beginner Assessment

A comprehensive, interactive web-based assessment designed for students and professionals starting their journey in **DevOps Engineering** and **Cloud Computing**.

## 🚀 About the Project
This project is an automated, self-contained evaluation tool. It allows students to test their foundational knowledge across critical cloud-native domains, including **Linux Administration, Networking, Docker, Git, and Cloud Infrastructure**. 

This tool is designed to be a "Zero-Configuration" exam—simply open the HTML file in any modern web browser to start the assessment.

## 🎯 Key Features
*   **50-Question Assessment:** Covers high-frequency topics like Linux commands, Dockerfile syntax, HTTP status codes, and Git workflows.
*   **Interactive Navigator:** An easy-to-use grid to track progress and jump between questions.
*   **60-Minute Exam Timer:** Simulates a real-world pressure testing environment.
*   **Diagnostic Review:** Students receive immediate feedback with detailed rationales/explanations after submission.
*   **Email Dispatch:** Automated result submission feature to streamline instructor grading.
*   **No Backend Required:** Built purely with HTML, CSS (Tailwind), and JavaScript.

## 🛠 Tech Stack
*   **Frontend:** HTML5, CSS3, Tailwind CSS.
*   **Logic:** Native JavaScript (ES6+).
*   **Deployment:** Static-site compatible (Ready for GitHub Pages).

## 🚀 How to Use
1. Clone this repository or download the `DevOps_Grand_Mix_50_Exam.html` file.
2. Open the file in any standard web browser (Chrome, Firefox, Edge).
3. Enter your name and begin the assessment.
4. After submission, review your diagnostic results.

## 📦 Topics Covered
- **Linux Administration:** Permissions (`chmod`, `chown`), System Utilities (`grep`, `ls`, `df`), and Process Management.
- **Networking:** HTTP/HTTPS status codes, default ports, and loopback addressing.
- **Docker:** `Dockerfile` architecture, image building, and container lifecycles.
- **Version Control:** Git staging, committing, and branching basics.
